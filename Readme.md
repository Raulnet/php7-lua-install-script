PHP7-Lua-Install-Script
====

#### Version:
- [Lua](https://www.lua.org/) 5.3.4
- [Pecl-Lua](https://pecl.php.net/package/lua/2.0.5) 2.0.5

#### Requirement:
* php7-dev
* php-pear
* libreadline-dev
* libncurses-dev
* make

How to use:
```bash
 $ cd /tmp
 $ git clone https://github.com/raulnet/php7-lua-install-script.git 
 $ cd php7-lua-install-script
 $ sudo ./install.sh
```
